# Gustave — Free Aim Breaker (early)

## Win condition

Stack Luck to maximise crit-driven Burn application from **Critical Burn**, dump spare AP on Free Aim shots to Mark + Burn everything in sight, then unload a **Shatter** or **Overcharge** on a broken target.

## Rotation

| Turn | Action |
| --- | --- |
| 1 | *Marking Shot* — apply Mark early so subsequent hits scale |
| 2 | Spend AP — *Powerful* for an ally buff, or *Lumière Assault* to build charge |
| 3 | *Shatter* on a broken enemy, or *Overcharge* if you're sitting on a big Charge bank |
| filler | Free Aim shots — cheap, apply Burn via Critical Burn lumina |

## Stat priority

**Luck > Might > Agility**. Luck drives crit rate, which triggers the Burn stacks that carry the build. Defensive stats are deprioritised — early game Gustave dies in 2 hits regardless.

## Variants

- If Luck scaling weapon isn't dropping, fall back on any Might-scaling offensive weapon
- Swap *Recovery* in for *Strike Storm* if you need a safety valve against AoE turns

# Verso — Burn Stacker (endgame)

## Win condition

Verso doesn't nuke — he **enables Maelle's nuke**. With Cheater + Combo Attack stack + Painted Power, Verso pumps out 80+ Burn stacks in a single turn just from chained Free Aim + Follow Up hits. Maelle's *Burning Canvas* then consumes those stacks for the big number.

## Rotation

| Turn | Action |
| --- | --- |
| 1 | *Follow Up* → *Phantom Stars* chain, apply Mark via Marking Shots lumina |
| 2 | Free Aim shots — each stacks Burn via Critical Burn + Double Burn |
| 3 | Hand off to Maelle for the Burning Canvas detonation |

## Stat priority

**Agi > Luck (to 75 % crit) > Vitality > Might**. Agility for turn speed + the Combo Attack chain, Luck for crit-driven Burn application, Vitality because Verso dies easy in endgame, Might is deprioritised — Verso isn't the one doing the damage.

## Variants

- *Confuso* replaces *Corpeso* against Fire-Absorb bosses
- Inverted Affinity lumina becomes mandatory on Fire-Absorb fights

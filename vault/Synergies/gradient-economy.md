# Gradient economy

## How it works

Gradient normally builds at 1/turn. These three pictos push it to 2-3/turn:

- **Gradient Parry** — on a successful parry (active defense), +1 Gradient
- **Gradient Overcharge** — any skill that normally applies Powerful grants +1 Gradient
- **Charging Critical** — crits grant +1 Gradient

At high build synergy (parry + multi-hit + crit stacking), Gradient can hit 3 by turn 2, enabling back-to-back Gradient Attacks.

## When to pick

- **Long boss fights** where sustained damage matters more than burst
- **Characters without a signature burst** (Lune, Sciel)
- If the user plays **reactively** (dodge/parry) — without parries the Gradient Parry picto is dead weight
